Education Booster is clean and responsive theme that suitable for both education and creative businesses. Education Booster is fully Gutenberg and Classic editor compatible.

== Description ==
Education Booster is a powerful WordPress theme that specializes in education. Education Booster is a robust and reliable, colorful and youthful, clean and pristine, easy to use and readily responsive WordPress creative website theme for professionals, colleges, schools, universities or private educational institutions. Education Booster has a streamlined, practical design that is sure to attract many readers. For Demo: https://keonthemes.com/theme-demo?id=MTE2M3xlZHVjYXRpb24tYm9vc3RlcnxFZHVjYXRpb24gQm9vc3Rlcg= and more Detail: https://keonthemes.com/downloads/education-booster

= Does this theme support any plugins? =

Education Booster includes support for WooCommerce, Contact From 7 and Jetpack.
