== Description ==

WIP WooCarousel Lite allows you to create a product slider carousel for your WooCommerce website.

You can get the premium version, from the following link:

[https://www.themeinprogress.com/wip-woocarousel-woocommerce-slider-carousel/](https://www.themeinprogress.com/wip-woocarousel-woocommerce-slider-carousel)

Install our WIP WooCarousel Lite, to show a carousel with your WooCommerce products.
Live demo: [http://demo.themeinprogress.com/woocarousel/free-version/](http://demo.themeinprogress.com/woocarousel/free-version)

= Installation =

Once you have installed the plugin, you just need to activate the plugin in order to enable it.

= Configuration =

WIP WooCarousel Lite will add a new admin page, where you can manage the plugin.

You need to insert the shortcode [wip_woocarousel_products_carousel] or use the shortcode generator from the WordPress editor:

Below the shortcode configuration.

product_items: Add -1 to display all products or insert the number of products to display
product_bestseller: Set this option ON, to display the best seller products for first
product_columns: Set the number of colums for this carousel.
product_rating: Set this option ON, to display the product rating
product_dots: Set this option ON, to display carousel dots

[wip_woocarousel_products_carousel product_items=”-1″ product_bestseller=”off|on” product_columns=”1|2|3|4|5…” product_rating=”off|on” product_dots=”off|on”]

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/wip-woocarousel-lite' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
